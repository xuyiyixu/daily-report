# 2026-03-03 Memory

## 日报项目

用户想要一个自动更新的每日热点日报网站，已搭建完成。

### 项目位置
`~/.openclaw/workspace/daily-report/`

### 日报规格（最终确认）
- 三个板块：AI科技、国际形势、产品创业
- 每个板块 5 条新闻
- 每个板块至少 1 条关于中国或来自中国媒体（新华社、人民日报、财新、36氪、澎湃新闻、环球时报等）
- 其余 4 条使用国际权威媒体（BBC、纽约时报、华尔街日报、金融时报、TechCrunch、The Verge、Crunchbase、彭博社等）
- 路透社可用可不用
- 优先主流媒体，避免小道消息
- 如果网页打不开，跳过该新闻并告知用户

### 推送设置
- 每天早上 9 点自动生成并通过 Feishu 推送
- HEARTBEAT.md 已配置好任务

### 技术细节
- 数据存储：`daily-report/data/YYYY-MM-DD.json`
- 网页：`daily-report/index.html`（本地 http://localhost:8000）
- JSON 中不能使用中文引号（会导致解析错误）

### 用户偏好
- 不要路透社（可选用）
- 不要小道消息
- 新闻要有原文链接
