# AI·创业·世界 - 每日热点

自动展示 AI科技、国际形势、产品创业的每日热点新闻。

## 🚀 最简单的使用方式

### 每天更新内容

直接在 Feishu 中给我发消息：

```
生成今日日报
```

我会自动：
1. 搜索三个板块的最新热点
2. 整理成中文标题和摘要
3. 更新 `data/YYYY-MM-DD.json` 文件
4. 网站自动显示最新内容

### 查看网站

```bash
cd daily-report
python3 -m http.server 8000
```

访问 http://localhost:8000

## 自动化方案

### 方案 A：每天提醒（推荐）

编辑 `~/.openclaw/workspace/HEARTBEAT.md`：

```markdown
# 每天早上 8 点检查日报
- 检查今天的日报是否已生成（daily-report/data/YYYY-MM-DD.json）
- 如果不存在，自动生成今日日报
- 生成后可以发送 Feishu 通知
```

### 方案 B：手动触发

每天在 Feishu 中发消息：
```
生成今日日报
```

### 方案 C：定时任务

```bash
crontab -e
```

添加（每天早上 8 点提醒）：
```
0 8 * * * echo "生成今日日报" | openclaw chat --channel feishu
```

## 数据格式

`data/YYYY-MM-DD.json`：

```json
{
  "date": "2026-03-02",
  "generated": "2026-03-02T14:00:00.000Z",
  "ai": [
    {
      "title": "新闻标题",
      "snippet": "简短摘要（50字以内）",
      "url": "https://原文链接"
    }
  ],
  "world": [...],
  "startup": [...]
}
```

## 工作原理

1. 你在 Feishu 中发消息："生成今日日报"
2. 我使用 `web_search` 搜索三个板块的热点
3. 整理成结构化的 JSON 数据
4. 保存到 `data/YYYY-MM-DD.json`
5. 网页自动加载并展示

**完全自动化，无需手动编辑代码！**

## 自定义

### 修改样式
编辑 `index.html` 中的 `<style>` 部分

### 修改板块
在 Feishu 中告诉我你想要的板块，我会帮你调整

### 修改新闻数量
默认每个板块 5 条，可以随时调整

## 故障排查

**问题：页面显示"今日报告尚未生成"**
- 在 Feishu 中发消息："生成今日日报"

**问题：历史记录不显示**
- 我会在生成日报时自动更新索引

**问题：新闻质量不好**
- 告诉我你的偏好，我会调整搜索关键词
